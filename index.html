Setup Supabase

Styleys India LLP ke liye permanent database connect karo. Ek baar setup karo, hamesha ke liye data save rahega.

Step 1: supabase.com pe free account banao
Step 2: New Project banao → naam: styleys-inventory
Step 3: SQL Editor mein yeh run karo:

CREATE TABLE items (id bigserial primary key, sku text, name text, category text, qty numeric, threshold numeric, unit text, price numeric, location text, created_at timestamptz default now());

CREATE TABLE sales (id bigserial primary key, order_no text, customer text, sale_date date, total numeric, gst numeric, created_at timestamptz default now());

CREATE TABLE sale_items (id bigserial primary key, sale_id bigint references sales(id), item_id bigint references items(id), item_name text, qty numeric, price numeric, unit text);

ALTER TABLE items ENABLE ROW LEVEL SECURITY;
ALTER TABLE sales ENABLE ROW LEVEL SECURITY;
ALTER TABLE sale_items ENABLE ROW LEVEL SECURITY;
CREATE POLICY "allow all" ON items FOR ALL USING (true) WITH CHECK (true);
CREATE POLICY "allow all" ON sales FOR ALL USING (true) WITH CHECK (true);
CREATE POLICY "allow all" ON sale_items FOR ALL USING (true) WITH CHECK (true);

Step 4: Settings → API → URL aur anon key copy karo
